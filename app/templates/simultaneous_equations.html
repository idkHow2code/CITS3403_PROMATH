Simultaneous Equations



Simultaneous equations are a set of equations involving variables where more than one variable is unknown. Depending on the number of unknown variables, there needs to be an equal number of different equations. In the previous tutorial we learned how to manipulate algebraic equations, and this skill is essential to have in order to solve these problems.


In this tutorial, we will be focusing only on linear simultaneous equations. There are essentially 4 different methods at your disposal to solve a set of simultaneous equations:


1. Addition
2. Subtraction
3. Multiplication
4. Substitution


The way to go about solving equations with any of these methods is to try and get everything in terms of one variable, which you can then solve for, and afterwards use the known variable(s) to work backwards and solve for other variables. Let's start with an example.



Example 1:


We have 2 linear equations


2x + 5y = 27
2x + 2y = 12

Now you will notice that in both equations there is a 2x variable, which means we can use the subtraction method to get rid of the x terms. To use the subtraction method, we simply subtract one equation from the other, like so:

Now you will notice that in both equations there is a 2x variable, which means we can use the subtraction method to get rid of the x terms. To use the subtraction method, we simply subtract one equation from the other, like so:


(2x + 5y) - (2x + 2y) = 27 - 12

Now we can perform the subtraction operation on both sides and see what's left


2x - 2x + 5y - 2y = 27 - 12
3y = 15

Now we can manipulate this equation to find y, and find that y = 5
Since we know the value for y, we can substitute it into any of the above equations to find x


2x + 5y = 27
2x + 5(5) = 27
2x + 25 = 27
2x = 2
x = 1

And there we have it, we have found both x and y (1 and 5)

Example 2:


We have 2 linear equations


3x + 3y = 21
6x - 4y = 22

When we multiply these equations, the aim is to do so in such a way where either the x or y terms have the same coefficient in each equation. Lets try multiplying the first equation by 2.


2(3x + 3y = 21) = 2(21)
6x + 6y = 42

Now this new equation has identical values of x and y as the first one, however the coefficients are in a state where we can use the subtraction method, like so: We take the first equation and subtract the second equation


(6x + 6y) - (6x - 4y) = 42 - 22

We now break this down and solve for y:


6x - 6x + 6y + 4y = 42 - 22
10y = 20
y = 2

Since we now have y, we can once again substitute y into any equation to produce x:


6x - 4y = 22
6x - 4(2) = 22
6x - 8 = 22
6x = 30
x = 5

And there we have it, we have found both x and y (5 and 2).

It is also possible to multiply both of the equations by different values in order to get the coefficients to match.
We have now seen the multiplication, as well as the addition and subtraction methods, which leaves the substitution method.

The substitution method is quite simple, and it starts the same way as the other methods - by getting everything in terms of one variable.


Example 3:


We have 2 linear equations


x + y = 19 2x + 4y = 46

We manipulate one of the equations to get it in terms of one variable, we will use equation 1.


x = 19 - y

Now that we have x in terms of y, we substitute the value for x into the other equation.


2(19 - y) + 4y = 46

Now we simplify to find y.


38 - 2y + 4y = 46
2y + 38 = 46
2y = 8
y = 4

Now we place y back into one of our first equations to find x.


x = 19 - (4)
x = 15

Now we have solved for x and y (15 and 4).
