Indices



Indices are called powers. It can be used to express a result of a number being multiplied multiple times.


For example, number 53 is basically having three 5s multiplied together, i.e. 5 x 5 x 5, and it evaluates to 125.


In the example above, "5" is called the base and "3" is the index.


In this section, we will learn the Law of Indices for manipulating indices.



Indices Rule 1: 0 power of any number equals to 1.


Exception:In this case, the base cannot be 0, as 00 cannot be evaluated.

i.e.

a0 = 1



Indices Rule 2: If the index of base a is -m, where m is a positive number, then it can be evaluated to be the inverse of a to the power of m


Exception:In this case, the base cannot be 0, as we cannot have a denominator to be 0.

i.e.

a-m = 1/am



Indices Rule 3: If two numbers with the same base are multiplied together, then we simply keep the base and add up the indices


i.e.

am x an = a(m+n)



Indices Rule 4: If two numbers with the same base are divided, then we simply keep the base and subtract the indices


i.e.

am / an = a(m-n)



Indices Rule 5: If an index is raised to another index, then we multiply the two indices


i.e.

(am)n= a(mxn)
